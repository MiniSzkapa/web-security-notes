# Authentication Vulnerabilities

## Co to jest
Błędy logowania i autoryzacji.

## Typy
- brute force
- weak passwords
- username enumeration
- broken sessions
- auth bypass

## Jak testować
- login form
- forgot password
- session cookies
- MFA bypass

## Co sprawdzać
- rate limiting
- password policy
- token invalidation

## Jak naprawić
- MFA
- rate limiting
- secure sessions
